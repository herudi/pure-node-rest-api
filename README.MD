# Pure Nodejs REST APIs

## Usage
```bash
git clone https://github.com/herudi/pure-node-rest-api.git
cd pure-node-rest-api
npm install
// or
yarn
```

## Route Available
Base Url : http://localhost:3000/
```js
GET /items
GET /items/:id
POST /items
PUT /items/:id
DELETE /items/id
```

## Sample Data Items
```json
{
    "name": "Nasi Goreng",
    "price": 12000,
    "brand": "Nasi"
}
```
